UBUNTU10.04的VIM配置文件为.vimrc，放在$(HOME)目录下，我的.vimrc内容如下：

```
set nocompatible
set ls=2
set tabstop=4
set shiftwidth=4
set showcmd
set hlsearch
set number
set autoindent
set smartindent
set cindent
```
